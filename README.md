# JasKleinanzeigenBot

Ein Kleinanzeigen Bot der alle 5 minuten nach den gefilterten Anzeigen sucht und diese dir per Telegram schickt. Dafür braucht man einen Telegram Bot Token und den bekommt man bei dem BotFather. 
Empfohlenes Video: https://www.youtube.com/watch?v=UQrcOj63S2o

# Setup:

Füge den BOTTOKEN in "BotToken.txt" ein
Führe die jar file aus!


# Commands:


/add "link": Fügt einen Link hinzu 


/list: Listet die Links auf


/delete "zahl": Löscht man einen Link aus /list
